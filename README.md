upyun for emlog 3.1.0 （又拍云图片上传插件）
===========================================

###插件简介：
把图片上传到又拍云存储，从而减轻服务器压力，CDN加速图片显示   
上传使用upyun的表单API功能，避免了客户服务器的文件中转，减少了不必要的流量，同时自动选择最近最快的节点接收客户端上传的文件，文件上传速度更快

###插件功能：
1，上传图片到又拍云存储   
2，查看，删除上传到又拍云存储图片   


###使用方法：
1.又拍云申请账号   
2.创建服务并添加授权   
3.功能配置开启表单API   
4.云处理--自定义版本180100   
   

1.emlog后台安装插件   
2.开启插件   
3.在又拍云设置填入对应的信息   


图文请看：http://www.xuejiajun.net/post-399.html
